# Cloud Infrastructure Automation Hub

Welcome to the Cloud Infrastructure Automation Hub. This repository serves as a centralized control center for managing, provisioning, and automating your cloud infrastructure across multiple providers.

## Features
* **Multi-Cloud Support:** Templates and modules for AWS, GCP, and Azure.
* **State Management:** Automated handling of infrastructure state files.
* **Disaster Recovery:** Built-in scripts for automated backups and failover processes.

## Getting Started
1. Clone the repository.
2. Configure your cloud provider credentials in `.env`.
3. Run `make init` followed by `make apply`.
